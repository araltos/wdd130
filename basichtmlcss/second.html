HTML Basics

What is HTML?

HTML, which stands for HyperText Markup Language, is the backbone of every webpage on the internet.
It's a language that helps browsers understand how to structure and present content.
Don't worry if that sounds technical â€“ we'll break it down step by step!
Your First HTML Page
Imagine your website as a house. HTML is the blueprint that defines the structure of your house.
Let's create the most basic HTML structure for your webpage:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Website</title> </head> <body> <h1>Welcome to My First Website!</h1> <p>This is a simple webpage created with HTML.</p> </body> </html>

<!DOCTYPE html>: This tells the browser that you're using HTML5, the latest version of HTML. <html lang="en">: The root element of your HTML document, indicating the language (in this case, English). <head>: Contains meta-information about your webpage, such as the character set and viewport settings. <title>: Sets the title of your webpage, which appears in the browser tab. <body>: This is where the content of your webpage goes, like text, images, and more.

Adding Content
Let's add some content to your webpage.
Think of the <h1> as the main heading of your page, and the <p> as a paragraph of text.


<body>
<h1>Welcome to My First Website!</h1>
<p>This is a simple webpage created with HTML. It's a wonderful journey into the world of web development!</p>
<p>Let's explore the basics of HTML together.</p>
</body>

Creating a Simple List HTML allows you to create lists easily. Let's add a simple unordered list (bulleted list) to your page.

<body> <h1>Welcome to My First Website!</h1> <p>This is a simple webpage created with HTML. It's a wonderful journey into the world of web development!</p> <p>Let's explore the basics of HTML together.</p> <ul> <li>Learn about HTML tags</li> <li>Understand the structure of a webpage</li> <li>Create your own content</li> </ul> </body>

Making it Responsive To make your website look good on various devices (like phones and tablets), you can add a meta tag in the <head> section.

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Website</title> </head>

This meta tag sets the viewport to the device's width and scales the content accordingly. Final Thoughts Congratulations! You've just created the foundation of your first webpage with HTML. It's like assembling the rooms of your digital house. As you learn more, you'll be able to add styles, colors, and even interactive elements to make your website unique.

CSS Fundamentals

What is CSS?

CSS, or Cascading Style Sheets, is a language used to style the appearance of your website.
It determines how your HTML elements are presented on the screen.
Think of it as the fashion designer for your web content â€“ it defines colors, fonts, spacing, and layout.

The Basics of CSS
Selector and Declaration
CSS rules consist of selectors and declarations.
The selector targets HTML elements, and the declaration block contains the styles. Here's a simple example:
/* Selector: Targets all paragraphs */
p {
/* Declaration block: Styles for paragraphs */
color: #333; = Text color
font-size: 16px; = Text size
margin-bottom: 20px; = Spacing at the bottom
}

Colors Colors in CSS can be defined using names, HEX codes, RGB, or HSL values.
For example:
* Using a color name */
h1 {
color: red;
}
/* Using a HEX code */
p {
color: #008080; = Teal color
}

Fonts
You can set the font family for your text using the font-family property:
body {
font-family: 'Arial', sans-serif;
}

Box Model
CSS treats every HTML element as a box.
The box model includes properties like margin, padding, border, and width/height.
For example:
/* Adding padding to a div */
div {
padding: 10px;
}

Responsive Design
Responsive design ensures your website looks good on various devices.
Use media queries to adjust styles based on screen size:
/* Adjusting styles for smaller screens */
@media only screen and (max-width: 600px) {
body {
font-size: 14px; /* Decrease font size */
}
/* More styles for small screens... */
}

Putting it All Together
Now, let's combine these concepts to style a simple webpage:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>My Simple Website</title>
</head>
<body>
<h1>Welcome to My Simple Website!</h1>
<p>This is a basic introduction to CSS for creating responsive websites.
Feel free to explore and customize these styles as you learn more!</p>
</body>
</html>
 This HTML file incorporates the discussed CSS concepts to create a simple, responsive webpage.
